# Projet bibliothèque

## Requirements

* Java 8
* Maven

## Lancer le projet

1. Se rendre à la racine du projet : `your_path/bibliotheque`
2. Lancer la commande Maven :
    
    `mvn clean install`

3. Lancer le projet :

    `java -jar target/bibliotheque-0.0.1-SNAPSHOT.jar`
    
4. L'application est démarrée à l'adresse : `localhost:8082`
